# Clean Geometric Logo

Read this reference when `logo_style` is `clean-geometric`.

Create three repository-specific concepts from distinct metaphors supported by
the project. Keep each Mark simple and geometric without reusing one generic AI,
bracket, node, path, or letter symbol across unrelated projects.

Candidate requirements:

- use exact `viewBox="0 0 512 512"` and a transparent background;
- use at most 16 basic SVG geometry elements;
- use only `#2855d9`, `#202124`, and `none` for fill and stroke;
- exclude text, gradients, masks, filters, embedded data, scripts, animation,
  and external references;
- remain recognizable in color, monochrome inspection, and at 16px.

Avoid brains, robots, sparkles, neural-network heads, chat bubbles, glowing
circuits, and similarity to established brands. Use temporary SVG concepts only
as renderer input; deliver the approved Mark and Lockup as transparent PNGs.
