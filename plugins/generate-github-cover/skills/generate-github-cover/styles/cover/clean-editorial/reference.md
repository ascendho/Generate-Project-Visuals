# Clean Editorial Cover

Read this reference when `cover_style` is `clean-editorial`.

- Use a concise editorial headline, not a feature list.
- Keep the two Cover support lines short enough for the narrow secondary column.
- Use the wider Social Preview copy fields for longer supporting copy.
- Omit terminal punctuation from image headlines unless the user requests it.
- Preserve the white background, blue accent, charcoal identity, serif headline,
  restrained contour lines, and generous safe areas.
- Keep Latin, Cyrillic, and Greek headlines italic. Render CJK and other scripts
  upright, and preserve the mirrored RTL layout.
- Do not add screenshots, gradients, remote images, or image-model artwork unless
  the user explicitly requests a different visual direction.

The templates define a `2000x400` Cover, a `2048x1024` Social Preview, and a
`1920x1080` Promo composition. The renderer exports only their fixed-size PNGs.
